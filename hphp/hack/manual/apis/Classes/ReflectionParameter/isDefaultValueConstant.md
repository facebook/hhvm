``` Hack
public function isDefaultValueConstant(): ?bool;
```




Returns TRUE if the default value is constant, FALSE if it is not or
NULL on failure.




## Returns




+ ` bool ` - If parameters default value is constant, or NULL if
  a default value does not exist.
