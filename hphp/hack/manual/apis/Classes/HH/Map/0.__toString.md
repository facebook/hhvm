Returns the ` string ` version of the current [` Map `](/apis/Classes/HH/Map/), which is `` "Map" ``




``` Hack
public function __toString(): string;
```




## Returns




+ ` string ` - The `` string `` ``` "Map" ```.




## Examples




The string version of a [` Map `](/apis/Classes/HH/Map/) is always `` "Map" ``:




``` basic-usage.hack
$m = Map {'a' => 1, 'b' => 2, 'c' => 3};
echo $m->__toString()."\n";

$m3 = Map {};
echo $m3->__toString()."\n";
```
