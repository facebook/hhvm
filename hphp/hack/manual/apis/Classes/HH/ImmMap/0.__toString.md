Returns the ` string ` version of the current [` ImmMap `](/apis/Classes/HH/ImmMap/), which is `` "ImmMap" ``




``` Hack
public function __toString(): string;
```




## Returns




+ ` string ` - The `` string `` ``` "ImmMap" ```.




## Examples




The string version of an [` ImmMap `](/apis/Classes/HH/ImmMap/) is always `` "ImmMap" ``:




``` basic-usage.hack
$im = ImmMap {'a' => 1, 'b' => 2, 'c' => 3};
echo $im->__toString()."\n";

$im3 = ImmMap {};
echo $im3->__toString()."\n";
```
