Returns the ` string ` version of the current [` ImmVector `](/apis/Classes/HH/ImmVector/), which is
`` "ImmVector" ``




``` Hack
public function __toString(): string;
```




## Returns




+ ` string ` - The `` string `` ``` "ImmVector" ```.




## Examples




The string version of a [` Vector `](/apis/Classes/HH/Vector/) is always `` "Vector" ``:




``` basic-usage.hack
$v = Vector {1, 2, 3};
echo $v->__toString()."\n";

$v2 = Vector {'a', 'b', 'c'};
echo $v2->__toString()."\n";

$v3 = Vector {};
echo $v3->__toString()."\n";
```
