Returns the ` string ` version of the current [` Pair `](/apis/Classes/HH/Pair/), which is `` "Pair" ``




``` Hack
public function __toString(): string;
```




## Returns




+ ` string ` - The `` string `` "Pair".




## Examples




The string version of a [` Pair `](/apis/Classes/HH/Pair/) is always `` "Pair" ``:




``` basic-usage.hack
$p = Pair {-1, 5};
echo $p->__toString()."\n";

$p2 = Pair {'foo', 'bar'};
echo $p2->__toString()."\n";
```
