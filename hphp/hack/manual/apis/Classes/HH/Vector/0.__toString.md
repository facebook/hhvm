Returns the ` string ` version of the current [` Vector `](/apis/Classes/HH/Vector/), which is `` "Vector" ``




``` Hack
public function __toString(): string;
```




## Returns




+ ` string ` - The string `` "Vector" ``.




## Examples




The string version of an [` ImmVector `](/apis/Classes/HH/ImmVector/) is always `` "ImmVector" ``:




``` basic-usage.hack
$iv = ImmVector {1, 2, 3};
echo $iv->__toString()."\n";

$iv2 = ImmVector {'a', 'b', 'c'};
echo $iv2->__toString()."\n";

$iv3 = ImmVector {};
echo $iv3->__toString()."\n";
```
