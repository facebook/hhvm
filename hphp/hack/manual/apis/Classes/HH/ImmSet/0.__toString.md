Returns the ` string ` version of this [` ImmSet `](/apis/Classes/HH/ImmSet/), which is `` "ImmSet" ``




``` Hack
public function __toString(): string;
```




## Returns




+ ` string ` - The `` string `` ``` "ImmSet" ```.




## Examples




The string version of an [` ImmSet `](/apis/Classes/HH/ImmSet/) is always `` "ImmSet" ``:




``` basic-usage.hack

$is2 = ImmSet {'a', 'b', 'c'};
```
